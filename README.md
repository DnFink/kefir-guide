# Milk Kefir Simple Guide

A clean, responsive, and mobile-friendly workflow guide for managing milk kefir grains. This project simplifies complex fermentation instructions into easy-to-follow daily, weekly, and vacation routines.

---

## Live Demo

You can view the live guide here:  
https://dnfink.github.io/kefir-guide/

---

## Features

- **Daily Cycle**: A step-by-step 24-hour routine for straining and refreshing grains.  
- **Weekly Maintenance**: Reminders for hygiene (jar swapping) and grain management (thinning).  
- **Vacation Mode**: Clear instructions on how to safely pause your fermentation for up to 3 weeks.  
- **Mobile First**: Designed with Tailwind CSS to look great on phones, tablets, and desktops.  
- **Zero Dependencies**: A single-file HTML solution that works offline and requires no complex setup.

---

## How to Use

1. **Clone or Download**  
   Save the `index.html` file to your computer.

2. **Open**  
   Double-click the file to open it in any web browser.

3. **Host on GitHub Pages**  
   - Create a new repository named `kefir-guide`  
   - Upload the `index.html` file  
   - Go to **Settings > Pages** and enable hosting from the `main` branch

---

## Maintenance Tips

- **Grains**: Always keep your grains in glass containers.  
- **Tools**: Use plastic or wood tools; avoid metal as it can react with the acidic environment of the kefir.  
- **Temperature**: Best fermented at room temperature (around 68–75°F).
